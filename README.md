# Student_Dashboard
Data Analysis project using Microsoft excel which is about students placement by their Institute.
📊 Student Placement & Salary Analysis | MS Excel Project
An end-to-end data analysis project built in Microsoft Excel, exploring placement outcomes and salary trends across 100+ engineering students from multiple branches and college tiers.
What it covers:

Student profiles across branches (CSE, IT, ECE, EEE, Mechanical, Civil) with tier-wise college classification
Key factors: CGPA, Python, DSA, ML & Web Dev skills, coding scores, communication, aptitude, internships, and projects
Placement outcomes across MNCs, Mid-size companies, Startups, and Top Tech firms
Salary analysis (LPA) by job role — Software Engineer, Data Scientist, Analyst & Web Developer

Skills demonstrated: Data cleaning · Pivot Tables · Conditional Analysis · Data Visualization · Multi-variable Insight Generation
